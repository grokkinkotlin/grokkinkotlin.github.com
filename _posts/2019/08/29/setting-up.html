About Kotlin

Kotlin is a Programming Language which supports both object oriented and functional programming paradigms. Kotlin can run on many platforms - JVM being one of them. It can be compiled to Javascript or native code (for noobs like me, it means it can run without a VM for e.g. embedded devices or IOS)

Hardware, OS and Shell

I’m using a MacBook Pro which is running MacOS Mojave (10.14.5)

Installation

We used Kotlin installation that comes with Jetbrains IntelliJ. But Kotlin Compiler is available to use with any standard text editor tools

IDE

We are JetBrains folks, from where Kotlin started and there is a good online support community around this.
